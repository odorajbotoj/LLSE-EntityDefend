# LLSE-EntityDefend

LLSE-EntityDefend 在一定区域内保护实体不受玩家攻击

## 安装

+ 放进 `plugins/` 即可

## 使用

+ 仅OP玩家可执行

+ `/ende add <from> <to> <name>` 添加一块保护区

+ `/ende rm <name>` 删除一块保护区

+ `/ende ls` 列出所有保护区

+ `/ende wl <name> <add|rm> <player>` 保护区白名单添加/删除玩家

+ `/ende en <name> <add|rm> <type>` 保护区保护名单添加/删除实体类型
